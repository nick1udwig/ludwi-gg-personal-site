This is an example blog post. The particle simulation on the home page demonstrates how physics can create beautiful, emergent patterns from simple rules.

Each particle is attracted to its target position by a spring force, while Brownian motion adds random thermal fluctuations. The temperature slider controls the intensity of these fluctuations - higher temperature means more chaos, lower temperature means more order.

The simulation uses Verlet integration for stable physics and TypedArrays for performance. On a modern device, it can handle thousands of particles at 60fps.
